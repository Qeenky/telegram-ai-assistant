**AI Ассистент в Telegram** с подписками\
Проект представляет собой шаблон для создания AI-ассистента в Telegram с системой монетизации через подписки. Ассистент использует DeepSeek API для генерации ответов и Yookassa для обработки платежей.

**Предварительные требования:**\
Python 3.10+\
PostgreSQL 14+\
Аккаунт в Yookassa для приема платежей\
API ключ от DeepSeek

**Команды**\
/buy - Купить premium подписку
\
/limit - Проверить текущий лимит токенов
\
/grant_premium <user_id> [days] - Выдать подписку (админ)

**Фоновые задачи**\
Сброс дневных лимитов: Ежедневно в 00:00 UTC
\
Проверка подписок: Каждую минуту
